Propósito del sitio

El propósito de realizar este sitio es reconocer e identificar las etiquetas básicas de HTML y de igual manera conocer sus respectivos atributos y valores que se utilizan en cada creación de un sitio web de manera básica y, los cuales permitiran un mejor uso en cada diseño creado.



INTRODUCCION

HTML es un lenguaje informatico que forma la mayoria de paginas web y plataformas online; html no es considerado un lenguaje de programación debido a que no puede crear una funcionalidad dinámica, sin embargo los usuarios pueden crear y estructurar parrafos, enlaces y secciones usando elementos, etiquetas y atributos adecuadamente, es por ello que se debe manejar estos conceptos basicos relacionados al html, los cuales van a permitir un mejor desarrollo del mismo.

Por ello es importante conocer, identificar y manejar las etiquetas básicas de html, asi como los atributos y sus valores, siendo ésta la base para poder darle un mejor uso a las mismas en la creación de distintas páginas que se quieran crear; con estos conceptos básicos de html los usuarios podran escribir las etiquetas, los atributos y demás contenido con mejor precision dandole un mejor manejo a dichas herramientas.


Actualmente existen 142 etiquetas de html disponibles las cuales permiten crear varios elementos que son la estructura básica de html, sin embargo no todas son utilizadas y por ello se seleccionó las primordiales o básicas a utilizar dentro de un diseño web.

Los elementos que se manejan en html tienen atributos y estos a su vez tienen un valor especifico los cuales configuran o ajustan su comportamiento de diversas maneras para cumplir así los criterio de los usuarios.
