Propósito del sitio

El propósito de realizar este sitio es reconocer e identificar las etiquetas básicas de HTML y de igual manera conocer sus respectivos atributos y valores que se utilizan en cada creación de un sitio web de manera básica y, los cuales permitiran un mejor uso en cada diseño creado.

Etiquetas Básicas de HTML

Según Prescott, P (2015), Las etiquetas básicas de HTML son códigos que sirven para dar instrucciones al navegador web sobre como mostrar el texto, es decir indica al navegador donde empieza y donde termina cada elemento.

Para el caso podemos mencionar las siguientes Etiquetas:

  • DOCTYPE HTML: Indica al navegador qué, es un documento HTML
  • html /html: Son las etiquetas de apertura y cierre donde irá todo el documento tanto la cabecera como el cuerpo del mismo.
  • head /head: Especifica la cabecera de todo el archivo
  • title /title: Este título se ubica en la barra de título junto al icono del programa
  • body /body: En esta etiqueta se ubica el cuerpo del documento
  • p /p: Esta etiqueta es para separar los párrafos que van dentro del documento
  • br /br: Indica un salto de linea, sería como un punto aparte
  • center /center: Centra todo aquello que esté dentro de la etiqueta
  • strong /strong: Convierte el texto en negrita
  • em /em: Establece el texto en cursiva
  • u /u: Subraya el texto
  • h /h: Esta etiqueta es la encargada de los encabezados y va desde h1que muestra un tamaño de letra grande, hasta h6 que su tamaño es más pequeño; estos encabezados permiten dividir el documento en secciones
  • ol li /li /ol: Sirve para crear listas ordenadas
  • ul li /li /ul: Sirve para crear listas sin orden
  • img /img: Permite mostrar imágenes dentro del contenido del documento
  • a /a: Permite crear un enlace a una página web junto al atributo href indica donde se pondrá el enlace y junto al atributo target se definirá si se abre en una nueva ventana o en la misma
  • div /div: Ayuda a estructurar el documento en secciones
  • span /span: Sirve para dar estilo a una parte del texto

Atributos y Valores

Los atributos son aquellas características que se le dan a un texto o imagen, como por ejemplo el ancho, el color, entre otros. Estos atributos van dentro de la etiqueta de inicio y van seguido del signo “igual” que va a declarar el valor del atributo, el cual va entre comillas. Por ejemplo:

  • Font color=”red” Hola /Font
    • Donde encontramos:
  • La etiqueta de apertura: Font
  • El atributo: color
  • Seguido el signo: =
  • El valor: “red”
  • El elemento: Hola
  • La etiqueta de cierre: /Font

Algunos atributos con sus respectivos valores serían:

    Para la etiqueta p /p cuya función es separar los párrafos del texto
  • alling=”right”: Indica alineación de párrafo a la derecha del documento
  • alling=”center”: Indica alineación de párrafo en el centro
  • alling=”left”: Indica alineación del párrafo a la izquierda
  • alling=”justify”: Justifica el párrafo

    • Para la etiqueta Font /Font que define características del texto
  • Face=”nombre_caracter”: cambia la fuente del texto
  • Color=”nombre color”: Cambia el color del texto
  • Size=”valor”: Cambia el tamaño del texto que puede ser desde 1 al 7

    • Para la etiqueta body /body: se puede asignar espacios de margen del documento, los cuales serian:
  • Leftmargin: Modifica margen de los lados de la página
  • Topmargin: Modifica margen de arriba y debajo de la página
    • El valor para estos atributos será expresado en pixeles o si quiere expresarlo en porcentaje debe introducir el signo del porcentaje al lado del número
